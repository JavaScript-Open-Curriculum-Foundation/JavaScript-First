About #JavaScriptFirst

Approach

This book attempts to adopt a Career and Technical Education appoach to the content development and implementation.

Objectives of the course

Teach anyone 16+ how to code a website, server and mobile app using JavaScript.

What this book covers:

This book is still in early access and open source. The content is made to be open source curriculum.

You will learn how to code with JavaScript using the node.js runtime environment rather than a browser and by the end, you will build a server and a website using JavaScript.

Content will ALWAYS be free for STUDENTS.

A live version of the first two chapters is available on twitch.tv/hansoncoding.

It takes a blended learning approach .

If the button ^ is live, click it.

About GitPod

GitPod is used to provide readers and students with a zero-config developer environment where they can follow along even on a chromebook or tablet.

This is helps provide an interactive experiance through codetogether, gitduck and other means of activities that facilitate remote learning.

Chapter 1: Reteaching JavaScript & Web Development

  • History of JavaScript
  • Learning Terminal, NVM, Git & Node.js
  • JavaScript Datatypes
  • logging datatypes
  • Types of Functions

Chapter 2: Introduction to Web Development with HTTP

  • Learn JavaScript before by building an HTTP Server with Node.js
  • Learn HTML with ES6 Template Strings
  • Secure your site with an SSL Certificate
  • Hosting your node.js server
  • Connect it to web Domain

Chapter 3: Test Driven Curriculum

  • Unit Tests
  • Users Tests (Twitter Bot)
  • Automatic Responsive Testing
  • Automatic Cross-Browser Testing
